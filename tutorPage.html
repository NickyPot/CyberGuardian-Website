HTML

Html was used for the websites main layout, images, and for the writing of the articles. It was also used to creat hypertext links to point to other points of the website.

CSS Framework: Bootstrap

The CSS framework Bootstrap 4 was used to give the website a responsive design. It enabled automatic scaling for different screen sizes and devices. It also helped with the quick implementation of features such as the slider and the navigation bar.

Custom CSS

Custom CSS was used to give the website a unique feeling. By just using the CSS elements that Bootstrap provides, the website would feel like any other, but after implemnting custom CSS elements I was able to achieve a result with character that can recognised as a unique website.

CSS Animation

With the help of bootstrap I was able to implement a carousel. A carousel is a slideshow of images that allows you to scroll through images that can also have links and captions attached to them.

PHP and MySQL

PHP and MySQL were utilized to implement a member system, where the member can login to access the information on the website.
